学习笔记

### 总结

#### 树

* 数的基本遍历方式
    * 前序 ：根左右
    * 中序 ：左根右
    * 后续 ：左右根
 * 常见的遍历实现方式：
     * 递归
     * 栈

#### 图

* 图的基本分类有
    * 无向无权图
    * 有向无权图
    * 无向有权图
* 常见算法
    * DFS
    * BFS


#### 堆

* 可以迅速找打一堆数中的最大或者最小的数据结构
* 基本复杂度(需要基本满足)
    * find-max : O(1)
    * delete-max : O(logN)
    * insert O(logN) O(1)
